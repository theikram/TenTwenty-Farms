## рҹҡҖ Getting Started

### Prerequisites

- **Node.js** >= 18.x
- **npm** >= 9.x

### Installation

```bash
# Clone repository
git clone https://github.com/theikram/TenTwenty-Farms.git

# Move into project
cd TenTwenty-Farms

# Install dependencies
npm install

# Start development server
npm run dev
```

The app runs at:

```bash
http://localhost:5173
```
